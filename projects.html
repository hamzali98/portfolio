SOFT Laundry

SOFT Laundry is a laundry management application built using flutter for front end and core PHP for backend. The application allows users to manage their laundry orders, view their orders, and track their laundry progress. The application is designed to be user-friendly and intuitive, with a clean and modern interface. The application is under development phase and will be completed soon.

RideNserve

RideNserve is a ride hailing application built using ionic with angular. In application user onboarding and driver onboarding modules are present. The application also have googleMaps integrated with only current location functionality. On boarding functionality is complete using node and Express server with MongoDB as database. The application is built for both Android and iOS devices. but checked only on android device. The application is no more in development phase because the development was shifted to flutter and assigned to another team.

Bookworm

A mobile app built with Flutter that allows users to browse, search, and save books. It provides a user-friendly interface for users to explore books, view book details, and save their favorite books. The app also includes a search feature that allows users to search for books by title, author, or genre.
